# System Approval — Company Knowledge Base

هذا الفولدر هو المرجع الرسمي لبيانات شركة HelioMax.
يُستخدم لتغذية المساعد الذكي "هيليو" بمعلومات الشركة.

## الملفات المتوقعة

| الملف | المحتوى |
|-------|---------|
| `products.md` | كتالوج المنتجات — VRF, Split, Cassette, Ducted |
| `pricing.md` | سياسة التسعير، الخصومات، العملات (USD / EGP / SAR) |
| `regions.md` | المناطق التي تعمل فيها الشركة (مصر / السعودية) |
| `teams.md` | هيكل الفرق — Sales, CS, Tech — ومسؤوليات كل فريق |
| `process.md` | سياسة المبيعات: مراحل البايبلاين، شروط التحويل، SLA |
| `faq.md` | أسئلة شائعة يسألها العملاء |
| `competitors.md` | منافسين رئيسيين وميزات HelioMax التنافسية |
